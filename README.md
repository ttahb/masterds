# masterds
Implementation of a few general purpose data structures and algorithms in plain JavaScript.

- Linked List
- Tree
- Stacks and Queues
- Graph
- Heap

Sorting
 - Bubble Sort
 - Selection Sort
 - Insertion Sort
 - Merge Sort
 - Quick Sort
 - Shell Sort
 - Heap Sort


Algorithms
- Recursion
- Heapify
- BFS ( Tree / Graph) 
- DFS (Tree -InOrder, PreOder & PostOrder / Graph)
- Prim's usign adj matrix
- Kruskal's using edge list.
